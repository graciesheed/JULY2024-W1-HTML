Basic structure uses

HTML and CSS are markup languages. Markup languages are not programming languages.

There is no logic or processing of data in the lines of Python of Javascript within a HTML page. HTML is merely a representative of data/information. HTML documents should always include the language declaration, a header containing the metadata of the webpage, a title which is shown in the browser's title bar and the body of information that is visible on screen to the user. We can create headings, ordered and unordered lists and insert images, videos, games and other content into our code to make it visible in the content for the user.

Headers

Headers range in size from H1 to H6. H1 is the largest font type. You don't need to use a p or br tag to ensure spacing between your header and text. This is done automatically as part of the header tag. Can be used to have control over the spacing.

List of things I need to do today.

Create an ordered an unordered list of jobs. Ordered lists are numbered and unordered lists are bulleted and used for when the order does not matter.

  • Clean the bathroom
  • Sort out the menu for the week
  • Look at stuff for Sally's birthday
  • Finish all of the content I'm behind in

I should probably use an ordered list for this so I can list them in order of importance.
  1. Finish all of the content I'm behind in
  2. Sort out the menu for this week
  3. Clean the bathroom
  4. Look at stuff for Sally's birthday

Ordered lists are good if you are providing instruction; for example, steps in a recipe. Unordered lists are fine for listed information where a numbered format is not required.

You can use semantic HTML elements throughout your webpage. These help convey meaning about the content they enclose. This is helpful for browers and web developers.
Some examples are article, section, header, main, details, summary etc. These aren't visible to the user.
Another example is figure/figcaption - this specifies self-contained content, like images, and figcaption provides a caption. I have also noticed that these have a margin when you insert them too.
